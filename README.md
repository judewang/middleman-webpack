# middleman-webpack

A middleman template using webpack to replace middleman asset pipeline.

## Requirements

- NodeJS
- Ruby
- Bundler installed
- npm or yarn installed

## Usage

Install pacakge

1. `bundle install`
2. `yarn` or `npm install`

Run develop server

    bundle exec middleman

Build production-ready code

    bundle exec middleman build
